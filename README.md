# Практична робота

## Використання Git і GitHub. Git Flow

### Мета роботи
Ознайомитися з основними можливостями системи контролю версій Git та сервісу GitHub, навчитися працювати з локальним і віддаленим репозиторіями, гілками, комітами та синхронізацією проєкту.

---

## Виконані завдання

### Git

* Створено локальний репозиторій.
* Виконано перший коміт.
* Створено гілки `develop` та `my-feature`.
* Додано зміни та створено коміти у відповідних гілках.
* Виконано злиття (`merge`) гілки `my-feature` у `develop`.
* Видалено гілку `my-feature`.
* Виконано злиття гілки `develop` у `main`.
* Видалено гілку `develop`.
* Переглянуто історію комітів.

### GitHub

* Створено обліковий запис GitHub.
* Створено віддалений репозиторій.
* Підключено локальний репозиторій до GitHub.
* Виконано відправлення (`push`) змін.
* Виконано отримання (`pull`) оновлень із віддаленого репозиторію.
